# Number Guessing Game (Python)
## Description
This is a simple number guessing game built using Python.
The computer randomly selects a number between 1 and 100, and the player tries to guess it.
## Technologies Used
- Python
- Random module
## Features
- User-friendly console interface
- Unlimited attempts
- Attempt counter
- Instant feedback (Too High / Too Low)
## Learning Outcome
- Understanding loops and conditions
- Working with random numbers
- Improving logical thinking
